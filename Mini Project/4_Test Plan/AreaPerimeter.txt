Test plan for the High Level Requirement Class Diagram UML:

Device modes = 2

Mode 1 : Area_Calculation      Mode 2: Perimeter_Calculation

Area_Calculation:

1. Only digits (0 or 1) is accepted
2. Only 2 shapes are accepted
	a. Circle
	b. Square
3. Data type : double (implicit conversion)
4. Unit of parameter = centimeter
5. Parameter > 0

Perimeter_Calculation:

1. Only digits (0 or 1) is accepted
2. Only 2 shapes are accepted
	a. Rectangle 
	b. Square
3. Data type = double (implicit conversion)
4. Unit of parameter = Centimeter
5. Parameter > 0
